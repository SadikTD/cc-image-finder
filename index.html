💡 Tips & Guide

🔍 Searching
Type any topic and press Enter. Suggestions appear as you type. Enable or disable sources using the chips below the search bar. Click "All" to toggle all sources at once.
⬇ Downloading
Click "Download & Resize" on any image. If focal point is enabled, you'll get to click where the face or important area is before cropping. Images are auto-resized to your configured dimensions (default 1200×675). Credit text is auto-copied to your clipboard.
📋 Using Credits
Every download auto-copies the proper attribution text. Open the Credits Log (📄 icon) to see all credits from this session. Use "Copy All" or "Export" to save them as a .txt file.
📍 Focal Point
When enabled in Settings, you can click on a face or key area before the crop happens. The image will be resized around your chosen center point so nothing important gets cut off.
🔲 Overlay Combiner
After setting the focal point, click "Add Overlay" to combine two images. Choose WGTC (orange ring) or AOTF (4-color ring), search for the overlay person, set their focal point, preview the result, and download. Both credits are saved automatically.
📝 Custom Filenames
Go to Settings → Filename to set a template. Variables: {query}, {title}, {source}, {creator}, {date}, {timestamp}. Default: "CC {title} - {source}" produces "CC President Donald Trump - wikimedia.jpg".
☑ Bulk Select
Click the checkbox on image cards (bottom-right corner) to select multiple images. Then use the toolbar to download all or copy all credits at once.
⭐ Favorites
Click the ★ icon on any image to save it to Favorites. Access them anytime from the ⭐ toolbar button. Favorites are saved in your browser.
🖱 Right-Click Menu
Right-click any image card for quick actions: download, copy credit, open source page, preview, favorite, or select.
⌨ Keyboard Shortcuts
Enter Search / Preview   Ctrl+K Focus search   ← → ↑ ↓ Navigate   D Download   C Copy credit   F Favorite   Space Select   Esc Close
🔑 API Keys
API keys are managed by the admin. Press Ctrl+Shift+A to unlock the API Keys tab in Settings. Wikimedia Commons works without any key. Keys are stored locally in your browser.
