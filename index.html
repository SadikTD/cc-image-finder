💡 Tips & Guide

🔍 Searching
Type any topic and press Enter. Suggestions appear as you type. Enable or disable sources using the chips below the search bar. Click "All" to toggle all sources at once.
⬇ Downloading
Click "Download & Resize" on any image. The visual editor opens — drag the crop box to position, drag the corners to resize/zoom, and use the sliders to adjust brightness, contrast, saturation, and sharpness. Hit "Crop & Download" when you're happy. You can disable the editor in Settings to skip straight to a center crop.
📋 Using Credits
Every download auto-copies the proper attribution text. Open the Credits Log (📄 icon) to see all credits from this session. Use "Copy All" or "Export" to save them as a .txt file.
✂️ Visual Editor
The crop box is locked to your output aspect ratio (default 1200×675). Drag the box to reposition, drag corners to resize. Sliders for brightness, contrast, saturation work in live preview. Sharpen applies on save. Use the Aspect Ratio buttons to lock to 16:9, 1:1, etc., or "Free" for any shape. Toggle the rule-of-thirds grid for composition help. Flip horizontal/vertical from the Transform section.
🖼 Collage Builder
Two ways to make a collage: (1) select 2 or 3 images on the main grid via the checkboxes, then click "🖼 Collage" in the bulk bar. (2) Open any image in the editor and click "🖼 Make Collage" to start with that image and search for more. Choose 2 or 3 horizontal slices, customize border color (default TFF red #DC2626), outer border thickness, divider thickness, and corner radius. Click any slice tab to edit its crop and adjustments individually. All credits get saved automatically.
📝 Custom Filenames
Go to Settings → Filename to set a template. Variables: {query}, {title}, {source}, {creator}, {date}, {timestamp}. Default: "CC {title} - {source}" produces "CC President Donald Trump - wikimedia.jpg".
☑ Bulk Select
Click the checkbox on image cards (bottom-right corner) to select multiple images. Then use the toolbar to download all or copy all credits at once.
⭐ Favorites
Click the ★ icon on any image to save it to Favorites. Access them anytime from the ⭐ toolbar button. Favorites are saved in your browser.
🖱 Right-Click Menu
Right-click any image card for quick actions: download, copy credit, open source page, preview, favorite, or select.
⌨ Keyboard Shortcuts
Enter Search / Preview   Ctrl+K Focus search   ← → ↑ ↓ Navigate   D Download   C Copy credit   F Favorite   Space Select   Esc Close
🔑 API Keys
API keys are managed by the admin. Press Ctrl+Shift+A to unlock the API Keys tab in Settings. Wikimedia Commons works without any key. Keys are stored locally in your browser.
🏛 Standalone Collage Maker
Click the purple Collage Maker button in the top bar to open an empty collage canvas. Choose 2 or 3 slices, then fill each slot by: searching (Search tab), pasting a screenshot (Paste tab → Ctrl+V), or uploading from your desktop (Upload tab → choose file). The same crop, pan, zoom, and adjustment controls apply to every slice.
